Política de Privacidade

Última atualização: 08 de março de 2026

Este aplicativo permite criar vídeos virais em segundos transformando qualquer ideia em vídeos prontos para TikTok, Reels ou Shorts utilizando tecnologias de inteligência artificial.

Informações coletadas

Podemos coletar informações fornecidas pelo usuário como textos, ideias ou prompts enviados para geração de vídeos.

Também podemos coletar dados técnicos como tipo de dispositivo, sistema operacional e dados de uso para melhorar a experiência do aplicativo.

Uso das informações

As informações coletadas são utilizadas para gerar vídeos, melhorar os serviços, corrigir erros e garantir o funcionamento adequado da plataforma.

Serviços de terceiros

O aplicativo pode utilizar serviços de inteligência artificial como Gemini e Veo para geração de conteúdo. Esses serviços podem processar temporariamente dados enviados para criação dos vídeos.

Compartilhamento de dados

Não vendemos ou alugamos dados pessoais. Informações podem ser compartilhadas apenas quando necessário para operar o serviço ou cumprir obrigações legais.

Segurança

Adotamos medidas razoáveis para proteger as informações contra acesso não autorizado ou uso indevido.

Direitos do usuário

Usuários podem solicitar a exclusão de dados ou entrar em contato para dúvidas sobre privacidade.
